Mini-Proyecto: Piedra, Papel o Tijera 🗿📄✂️

Felicidades por llegar a tu primer proyecto en la página. Aquí es donde todo lo que aprendiste en el Nivel 1 (variables, input, if/elif/else, y bucles) cobra vida. Vas a construir, paso a paso (te explicaré que tienes que hacer), el juego de Piedra, Papel o Tijera.

Paso 1: La Planificación (las recetas)

Antes de escribir código, piensa en lo que necesitas que haga el programa, y anótalas (aquí puedes omitir esto porque las recetas ya están, pero cuando tengas que hacer un proyecto real en tu U, trata de pensar en eso, ya que así piensan los programadores). Nuestra "receta" es:

  1. Darle la bienvenida al jugador.
  2. Empezar un bucle para que el juego continúe hasta que decidamos parar.
  3. Pedirle al jugador que elija entre "piedra", "papel" o "tijera".
  4. Hacer que la computadora elija una de las tres opciones al azar.
  5. Comparar las dos elecciones para ver quién gana.
  6. Mostrar el resultado (¡Ganaste!, ¡Perdiste! o ¡Es un empate!).
  7. Preguntar si el jugador quiere seguir jugando. Si dice "no", terminamos el juego.

Paso 2: Importar una Herramienta Mágica (`random`)

Para que la computadora elija al azar, necesitamos una "caja de herramientas" que ya viene con Python. Esta se llama random. La primera línea de nuestro código será para importarla y tenerla lista para usar. Este proceso de importar lo usarás demasiado en proyectos intermedios o avanzados, ya que facilita mucho las cosas.

import random  # Importamos la librería para generar aleatoriedad

Paso 3: El Bucle Principal y la Jugada del Usuario

Usaremos un bucle while True. Esto crea un ciclo infinito que solo se detendrá cuando se lo digas explícitamente. Dentro del bucle, pedirás la jugada al usuario con input(). Usarás .lower() para convertir su respuesta a minúsculas y evitar errores si escribe "Piedra" en lugar de "piedra".

while True:
    jugada_usuario = input("Elige piedra, papel o tijera: ").lower()

Paso 4: La Jugada de la Computadora

Ahora, le toca a la computadora. Crearás una lista con las opciones posibles y usarás random.choice() para que elija una de ellas (aquí es donde hacemos uso de la librería que importamos anteriormente).

    opciones = ["piedra", "papel", "tijera"]
    jugada_computadora = random.choice(opciones)
    print(f"La computadora eligió: {jugada_computadora}")

Paso 5: La Lógica de las Comparaciones (`if/elif/else`)

Esta es la parte central del juego. En el código haremos que se comparen las jugadas para determinar el resultado. Primero, el caso más fácil: el empate. Luego, todos los casos en los que gana el usuario. Si no es empate y tampoco gana el usuario, entonces, por descarte, ¡pierde!

    if jugada_usuario == jugada_computadora:
        print("Es empate!")
    elif (jugada_usuario == "piedra" and jugada_computadora == "tijera") or \
         (jugada_usuario == "tijera" and jugada_computadora == "papel") or \
         (jugada_usuario == "papel" and jugada_computadora == "piedra"):
        print("Ganaste!")
    else:
        print("Perdiste!😢")

Paso 6: ¿Otra Ronda?

Al final de cada ronda, le preguntaremos al usuario si quiere continuar. Si su respuesta no es "si", usaremos la palabra clave break para romper el bucle while True y terminar el programa.

    jugar_de_nuevo = input("¿Quieres jugar otra vez? (si/no): ").lower()
    if jugar_de_nuevo != "si":
        break

print("Gracias por jugar :)") # Este mensaje solo aparece al salir del bucle

El Código Completo

Aquí tienes todas las piezas juntas. ¡Pruébalo en Replit y mira cómo funciona!

import random

print("¡Bienvenida al juego de Piedra, Papel o Tijera!")

while True:
    # Pedimos la jugada al usuario
    jugada_usuario = input("Elige piedra, papel o tijera: ").lower()

    # Hacemos que la computadora elija
    opciones = ["piedra", "papel", "tijera"]
    jugada_computadora = random.choice(opciones)
    print(f"La computadora eligió: {jugada_computadora}")

    # Comparamos para determinar el resultado
    if jugada_usuario == jugada_computadora:
        print("Es empate!")
    elif (jugada_usuario == "piedra" and jugada_computadora == "tijera") or \
         (jugada_usuario == "tijera" and jugada_computadora == "papel") or \
         (jugada_usuario == "papel" and jugada_computadora == "piedra"):
        print("Ganaste!")
    else:
        print("Perdiste!😢")

    # Preguntamos si quiere jugar de nuevo
    jugar_de_nuevo = input("¿Quieres jugar otra vez? (si/no): ").lower()
    if jugar_de_nuevo != "si":
        break

print("Gracias por jugar :)")
