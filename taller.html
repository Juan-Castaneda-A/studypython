    Ejercicio de repaso 1: Área de un Triángulo Fácil

    Pistas Claves:

    • Necesitarás pedirle al usuario dos datos: la base y la altura.
    • Recuerda que la fórmula es: (base * altura) / 2.
    • Asegúrate de convertir las entradas del usuario a números (pueden ser decimales).

    Pseudocódigo

    Algoritmo AreaTriangulo
	Definir base,altura,area Como Real
	Escribir 'Introduce la base del triángulo: '
	Leer base
	Escribir 'Introduce la altura del triángulo: '
	Leer altura
	// 2. Calcular el área
	area <- (base*altura)/2
	// 3. Mostrar el resultado
	Escribir 'El área del triángulo es: ',area
FinAlgoritmo

    Diagrama de Flujo

    Código Python

    # 1. Pedir los datos al usuario
base = float(input("Introduce la base del triángulo: "))
altura = float(input("Introduce la altura del triángulo: "))

# 2. Calcular el área
area = (base * altura) / 2

# 3. Mostrar el resultado
print(f"El área del triángulo es: {area}")

    Ejercicio 1: Longitud de Circunferencia Fácil

    Enunciado: Diseñe un algoritmo que calcula y muestre la longitud de una circunferencia. Realice una implementación que use el radio y otra que use el diámetro. El algoritmo debe validar los datos.

    Pistas Claves:

    • La fórmula con radio es: Longitud = 2 * π * radio.
    • La fórmula con diámetro es: Longitud = π * diámetro.
    • El valor de Pi (π) es aproximadamente 3.14159.
    • Validar los datos significa asegurarse de que el radio o diámetro sean mayores que cero.

    Pseudocódigo

    Algoritmo ConvertirCelsiusAFahrenheit
	Definir radio,diametro,long Como Real
	Escribir 'Cálculo usando el radio:'
	Escribir 'Introduce el radio de la circunferencia:'
	Leer radio
	Si radio>0 Entonces
		long <- 2*PI*radio
		Escribir 'La longitud es: ',long
	SiNo
		Escribir 'El radio debe ser un número positivo.'
	FinSi
	Escribir '' // Línea en blanco para separar
	Escribir 'Cálculo usando el diámetro:'
	Escribir 'Introduce el diámetro de la circunferencia:'
	Leer diametro
	Si diametro>0 Entonces
		long <- PI*diametro
		Escribir 'La longitud es: ',long
	SiNo
		Escribir 'El diámetro debe ser un número positivo.'
	FinSi
FinAlgoritmo

    Diagrama de Flujo

    Código Python

    import math # Importamos la librería math para usar PI

# --- Versión con Radio ---
radio = float(input("Introduce el radio: "))

if radio > 0:
    longitud = 2 * math.pi * radio
    print(f"La longitud (con radio) es: {longitud}")
else:
    print("Error: el radio debe ser positivo.")

print("-" * 20) # Separador

# --- Versión con Diámetro ---
diametro = float(input("Introduce el diámetro: "))

if diametro > 0:
    longitud = math.pi * diametro
    print(f"La longitud (con diámetro) es: {longitud}")
else:
    print("Error: el diámetro debe ser positivo.")
    Ejercicio 3: Calcular Hipotenusa Intermedio

    Enunciado: Diseñe un algoritmo que calcula y muestre la longitud de la hipotenusa de un triángulo rectángulo, pidiendo el valor de sus dos catetos.

    Pistas Claves:

    • Usa el Teorema de Pitágoras: a² + b² = c².
    • Para encontrar la hipotenusa (c), necesitas calcular la raíz cuadrada de (a² + b²).
    • En Python, la raíz cuadrada se puede calcular con math.sqrt() o elevando a 0.5 (**(0.5)).

    Pseudocódigo

    Algoritmo ConvertirCelsiusAFahrenheit
	Definir catetoA, catetoB, hipotenusa Como Real
    
    Escribir "Introduce la longitud del primer cateto:"
    Leer catetoA
    Escribir "Introduce la longitud del segundo cateto:"
    Leer catetoB
    
    Si catetoA > 0 Y catetoB > 0 Entonces
        hipotenusa <- RC(catetoA^2 + catetoB^2)
        Escribir "La longitud de la hipotenusa es: ", hipotenusa
    SiNo
        Escribir "Las longitudes de los catetos deben ser positivas."
    FinSi
FinAlgoritmo

    Diagrama de Flujo

    Código Python

    import math

cateto_a = float(input("Introduce el cateto a: "))
cateto_b = float(input("Introduce el cateto b: "))

if cateto_a > 0 and cateto_b > 0:
    # Calculamos la suma de los cuadrados
    suma_cuadrados = (cateto_a ** 2) + (cateto_b ** 2)
    
    # Calculamos la raíz cuadrada
    hipotenusa = math.sqrt(suma_cuadrados)
    
    print(f"La hipotenusa es: {hipotenusa}")
else:
    print("Error: las longitudes deben ser positivas.")
    Ejercicio 4: Segundos para el Minuto Intermedio

    Enunciado: Diseñe un algoritmo que, para un cierto tiempo en segundos, calcule los segundos restantes que le correspondan para convertirse exactamente en minutos.

    Pistas Claves:

    • El operador módulo (% en Python, MOD en PSeInt) es la clave. Te da el residuo de una división.
    • segundos % 60 te dirá cuántos segundos "sobran" después de formar todos los minutos posibles.
    • Si sobran 40 segundos, ¿cuántos faltan para llegar a 60?
    • ¿Qué pasa si sobran 0 segundos (es un minuto exacto)?

    Pseudocódigo

    Algoritmo SegundosRestantes
    Definir segundos_totales, segundos_sobrantes, segundos_faltantes Como Entero
    
    Escribir "Introduce una cantidad de segundos:"
    Leer segundos_totales
    
    Si segundos_totales >= 0 Entonces
        segundos_sobrantes <- segundos_totales MOD 60
        
        Si segundos_sobrantes == 0 Entonces
            segundos_faltantes <- 0
        SiNo
            segundos_faltantes <- 60 - segundos_sobrantes
        FinSi
        
        Escribir "Faltan ", segundos_faltantes, " segundos para el siguiente minuto."
    SiNo
        Escribir "Los segundos no pueden ser negativos."
    FinSi
FinAlgoritmo

    Diagrama de Flujo

    Código Python

    segundos_totales = int(input("Introduce los segundos: "))

if segundos_totales >= 0:
    # Calculamos cuántos segundos 'sobran' del último minuto
    segundos_sobrantes = segundos_totales % 60
    
    if segundos_sobrantes == 0:
        # Si es un minuto exacto, no falta nada
        segundos_faltantes = 0
    else:
        # Si no, calculamos lo que falta para llegar a 60
        segundos_faltantes = 60 - segundos_sobrantes
        
    print(f"Faltan {segundos_faltantes} segundos para completar el siguiente minuto.")
else:
    print("Error: los segundos no pueden ser negativos.")
    Ejercicio 6: Número Par o Impar Fácil

    Enunciado: Diseñe un algoritmo que determine si un número entero ingresado es par o impar.

    Pistas Claves:

    • Un número es par si al dividirlo por 2, su residuo es 0.
    • El operador módulo (% o MOD) es perfecto para esto.

    Pseudocódigo

    Algoritmo ParImpar
    Definir numero Como Entero
    
    Escribir "Introduce un número entero:"
    Leer numero
    
    Si numero MOD 2 == 0 Entonces
        Escribir "El número ", numero, " es PAR."
    SiNo
        Escribir "El número ", numero, " es IMPAR."
    FinSi
FinAlgoritmo

    Diagrama de Flujo

    Código Python

    numero = int(input("Introduce un número entero: "))

# Si el residuo de dividir por 2 es 0, es par
if numero % 2 == 0:
    print(f"El número {numero} es PAR.")
else:
    print(f"El número {numero} es IMPAR.")

    Ejercicio 13: Etapas de la Vida Intermedio

    Enunciado: Diseñe un algoritmo que, al ingresar la edad de una persona, muestre el mensaje correspondiente a su etapa de vida (NIÑO, PUBER, ADOLESCENTE, JOVEN, ADULTO, ANCIANO).

    Pistas Claves:

    • Este es un caso perfecto para usar condicionales anidados o una cadena de SiNo Si... (elif en Python).
    • Define claramente los rangos: NIÑO (<=10), PUBER (11-14), ADOLESCENTE (15-18), JOVEN (19-25), ADULTO (26-65), ANCIANO (>65).
    • Valida que la edad no sea un número negativo.

    Pseudocódigo

    Algoritmo EtapasVida
    Definir edad Como Entero
    
    Escribir "Introduce la edad:"
    Leer edad
    
    Si edad >= 0 Entonces
        Si edad <= 10 Entonces
            Escribir "NIÑO"
        SiNo
            Si edad <= 14 Entonces
                Escribir "PUBER"
            SiNo
                Si edad <= 18 Entonces
                    Escribir "ADOLESCENTE"
                SiNo
                    Si edad <= 25 Entonces
                        Escribir "JOVEN"
                    SiNo
                        Si edad <= 65 Entonces
                            Escribir "ADULTO"
                        SiNo
                            Escribir "ANCIANO"
                        FinSi
                    FinSi
                FinSi
            FinSi
        FinSi
    SiNo
        Escribir "La edad no puede ser negativa."
    FinSi
FinAlgoritmo

    Diagrama de Flujo

    Código Python

    edad = int(input("Introduce la edad: "))

if edad < 0:
    print("Error: la edad no puede ser negativa.")
elif edad <= 10:
    print("NIÑO")
elif edad <= 14:
    print("PUBER")
elif edad <= 18:
    print("ADOLESCENTE")
elif edad <= 25:
    print("JOVEN")
elif edad <= 65:
    print("ADULTO")
else: # Si no es ninguna de las anteriores, es mayor de 65
    print("ANCIANO")
